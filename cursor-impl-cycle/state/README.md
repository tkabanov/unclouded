# cursor-impl-cycle state

Runtime artifacts for the implementation cycle. Created by `scripts/bootstrap.mjs`.

| File | Purpose |
|------|---------|
| `cycle.json` | Active phase, gate stage, current targets |
| `module-map.json` | Scope phase output |
| `item-registry.json` | Flattened implement items from decompose |
| `last-brief.json` | Current hook follow-up brief |
| `progress.txt` | Human-readable log |
| `briefs/` | Per-dispatch brief JSON files |

See `cycle.example.json` for the initial schema.
